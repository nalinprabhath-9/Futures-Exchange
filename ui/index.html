Quick start:
1) Run your setup shell script (creates users + starts nodes).
2) In Wallets, fund users (faucet) and lock collateral if your backend supports it.
3) In Proposals, create a proposal → accept it → mine a block.
4) In Mempool and Blockchain, verify gossip + inclusion.

Network Overview

This panel polls each node’s health, mempool size, and open proposal count. Green dot means node is reachable; red means request failed.
Tip: This UI uses a same-origin proxy (/api/:port) and can run your CLI scripts via /action/*. Use Node 5002 to accept, then watch Node 5001/5003 mempools to see gossip.
