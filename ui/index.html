Quick start
1) Run your setup script to start nodes and create users.
2) In Wallets, check balances and fund users if needed.
3) In Proposals, create a signed proposal.
4) Accept the proposal as another user.
5) In Mempool, verify transaction gossip.
6) In Blockchain, mine blocks and verify inclusion.
